# Мой магазин - это магазин канцелярских товаров

## Описание

В нашем магазине вы можете найти все для эстетичной и комфортной учебы, и даже для работы. На данный момент на нашем сайте реализовано три страницы с навигацией между ними, но дальше-больше. Рады видеть вас!

## Страницы магазина

| Главная страница | Содержит в себе ссылки на другие две страницы |
| Об авторе | Содержит краткую инфу о разработчике |
| О магазине| Пока содержит только тему нашей лабы, то есть то,что можно приобрести в нашем магазине |
